# CPP labs

Один Гитхаб репозиторий для всех лабораторных работ по курсам КР2, КР3 и КР4. Каждая практика имеет свою папку с `main.cpp` и `README.md` с отчетом.

## How to build

### Visual Studio

1. `File -> Open -> Folder...`
2. Выбери эту папку репозитория
3. Дождись завершения настройки CMake
4. Выберите необходимую работу, например, `kr4_pr11_fraction`.
5. Все, можно запускать.

## Формат отчета для каждой практики

Каждая практика должна иметь `README.md` с отчетом, который должен содержать следующие пункты:

1. Задача, которую нужно решить.
2. Номер варианта.
4. Тестовые данные.
5. Вывод программы.
