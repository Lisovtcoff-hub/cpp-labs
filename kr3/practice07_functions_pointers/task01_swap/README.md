# kr2 practice 0 -

## Текст задачи

Реализовать функцию swal для обмена значений между аргументами x и y, не
используя при этом третьей переменной. 

## Тесты

### Тест 1
Ввод:
```
4
5
```

Вывод:
```
До обмена:
x = 4
y = 5
После обмена:
x = 5
y = 4
```

### Тест 2
Ввод:
```
45452354
2342342342
```

Вывод:
```
До обмена:
x = 45452354
y = 2147483647
После обмена:
x = 2147483647
y = 45452354
```
