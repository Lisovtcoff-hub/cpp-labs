# kr2 practice 07 - task03_overload_min_max

## Текст задачи

Реализуйте перегрузку функций max и min для массива из k вещественных
чисел. С помощью программы найдите произведение минимумов двух
массивов различной длины и среднее геометрическое их максимумов.

## Тесты

### Тест 1
Ввод:
```
5
4
4 2 8 1 6
3 9 5 7
```

Вывод:
```
Минимум первого массива: 1
Минимум второго массива: 3
Максимум первого массива: 8
Максимум второго массива: 9
Произведение минимумов: 3
Среднее геометрическое максимумов: 8.48528
```

### Тест 2
Ввод:
```
4
3
4 -2 9 5
16 1 8
```

Вывод:
```
Минимум первого массива: -2
Минимум второго массива: 1
Максимум первого массива: 9
Максимум второго массива: 16
Произведение минимумов: -2
Среднее геометрическое максимумов: 12
```
