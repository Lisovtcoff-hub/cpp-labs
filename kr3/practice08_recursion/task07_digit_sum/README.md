# kr2 practice 08 - task07_digit_sum

## Текст задачи

Дано натуральное число N. Вычислите сумму его цифр. При решении этой
задачи нельзя использовать строки, списки (вектора), массивы, циклы

## Тесты

### Тест 1
Ввод:
```
123456789
```

Вывод:
```
45
```

### Тест 2
Ввод:
```
2147483647
```

Вывод:
```
46
```
